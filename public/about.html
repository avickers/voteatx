Find It Nearby

The "Find It Nearby" application will identify features in Austin, TX that are closest to you. The features it knows about are:

  • police station
  • fire station
  • post office
  • library
  • moon tower (huh?)

When you first start the application it will try to determine where you are. Depending on your web browser settings, you may be promoted to allow your location to be identified. Once you have been located, it will draw a map showing your position and the nearby features.

Here are some things you can do with the map:

  • Click at a location on the map to see what features are nearby.
  • Drag the red "you are here" marker around, to explore a different area.
  • Type an address in the field at the bottom of the screen to map that location.

Voting Place

The "Vote" icon identifies the election day voting place for the selected location.

The voting place data is for the November 6, 2012 general election.

This feature is possible thanks to the open data published by Travis County Clerk (voting places) and Travis County Tax-Assessor Collector (voting district).

Powered by Open Source and Open Government

In January 2012, the City of Austin released the data.austintexas.gov open government data portal. This application is built on publicly available data provided there. At this time, there are only a limited number of data collections available, so only a small set of features are supported here. What else would you like to see? How about police stations? Driver license offices? Public wi-fi access points? If the City releases that information, I'll add it to the application.
